# Fastfetch MOTD

Fastfetch config with ASCII art logos for a dynamic and colorful MOTD.

```sh
# Clone repository
git clone https://github.com/teanup/fastfetch-motd.git

# Copy config and MOTD script
cp fastfetch-motd/fastfetch-update-motd.sh /etc/update-motd.d/10-fastfetch
cp fastfetch-motd/fastfetch-peanut.jsonc /etc/update-motd.d/fastfetch.jsonc
```
